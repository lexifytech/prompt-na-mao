# Prompt na Mão — Landing Page

> Landing page profissional e de alta conversão da marca **Prompt na Mão**, focada na venda e distribuição de prompts de Inteligência Artificial estruturados para marketing, criação de conteúdo, vídeos e geração de imagens.

---

## 🚀 Sobre o Projeto

O **Prompt na Mão** é um ecossistema projetado para quem deseja economizar horas de trabalho utilizando inteligência artificial de forma inteligente. Com uma biblioteca com mais de 10.000 prompts profissionais validados, o objetivo é permitir que produtores de conteúdo, profissionais de marketing e criadores gerem resultados de alta qualidade em segundos.

Esta landing page foi desenvolvida com foco total em experiência do usuário (UX), carregamento rápido e otimização de conversão.

## 🛠️ Tecnologias Utilizadas

O projeto foi construído seguindo as melhores práticas de desenvolvimento front-end moderno utilizando tecnologias puras (Vanilla):

* **HTML5:** Estrutura semântica e acessível.
* **CSS3 (Custom Properties / Flexbox / Grid):** Estilização rica, responsiva, com suporte a efeitos modernos de gradientes, glassmorphism e animações fluidas.
* **JavaScript (ES6+):** Lógica de interatividade (FAQs interativos, contadores dinâmicos, controle de abas/filtros, sliders de depoimentos e ancoragem suave).
* **Google Fonts:** Tipografia moderna (Barlow Condensed e Inter) para legibilidade premium.

## 💻 Como Rodar o Projeto Localmente

1. Clone este repositório:
   ```bash
   git clone git@github.com:lexifytech/prompt-na-mao.git
   ```
2. Entre no diretório do projeto:
   ```bash
   cd prompt-na-mao
   ```
3. Abra o arquivo `index.html` em qualquer navegador web de sua preferência ou inicie um servidor local como o Live Server do VS Code.
